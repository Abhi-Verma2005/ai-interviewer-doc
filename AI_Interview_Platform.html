Prompt caching saves 82% of cost

Every scoring call uses the same 3,000-token rubric. We cache it after the first question and read it back at 10% of the price for every subsequent question.

The hallucination guard

Every score must include a verbatim quote from the candidate's actual words. If Claude paraphrases or invents a quote, the score is rejected and re-run.

✅ Accepted
// Quote is word-for-word from the transcript { "technical": { "score": 8, "quote": "I would use a hash map because lookup is O(1) average case" } }
❌ Rejected — re-run
// This is the AI's summary, not the candidate's words { "technical": { "score": 8, "quote": "The candidate explained the approach clearly" } }
Every score in your report has evidence you can verify. Click the quote, read the transcript, decide if you agree.
Section 05

The 5-Layer Anti-Cheat Defense

Cheating in AI-era hiring is real. Candidates use ChatGPT in another tab, have friends feeding answers, paste prepared code, or get someone else to take the interview entirely. All five layers run simultaneously.

1
Identity VerificationEnterprise tier · pre-interview
  • Photo ID upload compared against live webcam selfie
  • ID hashed immediately — image never stored
2
Liveness via MediaPipe
  • Number of faces in frame — more than one is flagged
  • Same face throughout — detects mid-interview swapping
  • 3D depth check — defeats flat photos or screen replays
  • Random blink/nod challenge every 4 minutes
3
Voice Biometrics
  • Baseline fingerprint captured in the warm-up 60 seconds
  • Continuous comparison — same voice throughout?
  • Background voice monitoring — catches off-camera coaching
  • Room acoustic signature — flags earpiece patterns
4
Keystroke DynamicsCode questions only
  • Typing speed and rhythm — natural flow vs sudden large paste?
  • Paste events — 200 lines appearing in 0.1 seconds is a signal
  • Tab switches — how long was the candidate on another window?
  • Clipboard hash compared against known solution patterns
5
AI-Generated Answer Detection
  • Pre-computed baseline: "what would GPT say?" for each question
  • Candidate answer compared against the AI reference
  • Low perplexity + no personal anecdotes + perfect generic grammar = elevated risk score
⚠️ Important — flags are not rejections
A flag means "take a look." Some candidates type fast. Some have perfect grammar. You make the call. We never automatically reject based on these signals.
